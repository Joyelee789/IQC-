1. 系統簡介
═══════════════════════════════════════════════════════════════

IQC 效率管理系統是一個用於分析檢驗人員效率、工作負載和時間分配的
數據管理工具。系統透過視覺化圖表和統計分析,協助提升 IQC 檢驗效能
與品質水平。

主要功能:
• 檢驗效率分析與監控
• 工作負載統計與分配
• 時間管理與分析
• MRB (Material Review Board) 追蹤
• 物料類別效率比較
• 額外任務時間記錄

3. 安裝與啟動
═══════════════════════════════════════════════════════════════

📦 安裝步驟:
   1. 解壓縮 "IQC效率管理系統.zip" 到任意資料夾
   2. 確保資料夾中包含以下檔案:
      - IQC效率管理系統.exe (主程式)
      - IQC1.png, IQC2.png, IQC3.png (系統圖片)
      - 使用說明.txt (本文件)

🚀 啟動程式:
   1. 雙擊 "IQC效率管理系統.exe"
   2. 等待 5-10 秒讓系統啟動
   3. 瀏覽器會自動開啟系統介面
   4. 如果瀏覽器沒有自動開啟,請手動訪問:
      http://localhost:8501

⚠️ 重要提醒:
   • 程式運行時請勿關閉黑色命令列視窗
   • 關閉瀏覽器分頁不會停止系統
   • 如需完全退出,請關閉命令列視窗

4. 使用方法
═══════════════════════════════════════════════════════════════

📂 步驟一: 上傳資料檔案
   1. 點擊左側邊欄的「資料上傳」區域
   2. 選擇並上傳以下 Excel 檔案:
      • IQC Report (必要)
      • PCB建檔明細 (PCB類別必要)
      • PCB標準工時對應表 (PCB類別必要)
      • IQC額外任務紀錄清單 (可選)
   3. 點擊「處理資料」按鈕

📊 步驟二: 查看分析結果
   系統會自動產生以下分析頁面:
   • 效率監控 - 檢視各檢驗員的效率表現
   • 工作負載 - 分析工作量分配情況
   • 時間管理 - 檢視時間使用分配
   • 額外任務監控 - 追蹤非檢驗任務時間

📅 步驟三: 篩選資料 (可選)
   1. 在左側邊欄選擇「開始日期」和「結束日期」
   2. 點擊「應用日期篩選」按鈕
   3. 系統會重新計算選定期間的數據

💾 步驟四: 匯出報表
   1. 點擊左側邊欄的「下載匯出Excel報表」按鈕
   2. 系統會自動下載包含所有分析結果的 Excel 檔案

5. 檔案格式要求
═══════════════════════════════════════════════════════════════

📄 IQC Report.xlsx
   必要欄位:
   • 檢驗員 (或 Inspector)
   • 檢驗日期 (或 Inspection Date)
   • 料號 (或 Part No.)
   • 類別 (或 Category)
   • 抽樣狀態 (或 Sampling Status)
   • 檢驗標準工時 (或 Standard Time)
   • 檢驗耗時（調整後）(或 Actual Time)

📄 PCB建檔明細.xlsx
   必要欄位:
   • 料號
   • 壓合孔數 (N欄,第14欄)
   • 版長
   • 版寬

📄 PCB標準工時對應表.xlsx
   必要欄位:
   • 面積範圍
   • 壓合總孔數
   • PCB標準工時

📄 IQC額外任務紀錄清單.xlsx
   必要欄位:
   • 姓名
   • 日期
   • 工作事項分類
   • 用時(分鐘)

6. 常見問題 (FAQ)
═══════════════════════════════════════════════════════════════

Q1: 程式啟動後瀏覽器沒有自動開啟?
A1: 請手動開啟瀏覽器並訪問 http://localhost:8501

Q2: 上傳檔案後出現錯誤?
A2: 請檢查:
    • Excel 檔案格式是否正確 (.xlsx 或 .xls)
    • 必要欄位是否都存在
    • 檔案是否被其他程式開啟
    • 資料格式是否符合要求

Q3: 處理資料時程式卡住?
A3: 這是正常現象,大量資料需要時間處理,請耐心等待

Q4: 圖表顯示不正常?
A4: 請嘗試:
    • 重新整理瀏覽器頁面 (F5)
    • 清除瀏覽器快取
    • 使用 Chrome 或 Edge 瀏覽器

Q5: 如何完全關閉程式?
A5: 關閉黑色命令列視窗即可

Q6: 可以同時處理多個月份的資料嗎?
A6: 可以,上傳多個 IQC Report 檔案即可

Q7: 匯出的 Excel 報表包含哪些內容?
A7: 包含所有已處理的分析資料、效率統計、工作負載等

Q8: 系統支援哪些語言?
A8: 目前支援繁體中文和英文欄位名稱

Q9: 為什麼有些檢驗員不在圖表中顯示?
A9: 系統會自動排除特定檢驗員 (如: 謝芷馨、蘇育珍、許碧琦)

Q10: 端口 8501 已被占用怎麼辦?
A10: 程式會自動尋找其他可用端口 (8502, 8503...)
